# Safe_Food

# 프로젝트 명: 
    Safe_food

# 프로젝트 설명:
    안전 먹거리에 대한 정보를 식품별 첨가물과 영양 성분을 데이터를 분석하고, 식품 별
    알레르기 유의사항을 표시하여 고객에게 안전 한 먹거리를 선택할 수 있도록 하고, 고객이
    섭취한 식품들을 통해 어는 정도의 영양분을 섭취했는지 분석결과를 보여주는 웹 프로젝트

# 아이디어 추가:
    사용자가 음식 사진으로 검색 할 수 있는 이미지 서치 기능을 추가하여 관련 음식의 레시피, 영양소 정보, 
    알레르기 정보를 알려준다
